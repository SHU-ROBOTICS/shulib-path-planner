# Seasons

Configuration files for each VEX game season.

## Creating a New Season

1. Create a folder: `seasons/your_season_name/`
2. Copy `_template.json` to `your_season_name/config.json`
3. Edit the config to set:
   - Which command categories to include
   - Any command overrides for your mechanisms
   - Custom commands specific to this season
   - Command sequences for common actions
   - Default starting positions
