# Push Back 2025-2026

## Game Overview

- **Scoring Objects:** 88 blocks (44 red, 44 blue), ~3.25" diameter
- **Goals:** 2 long goals (15 capacity), 2 center goals (7 capacity)
- **Key Mechanic:** Goals overflow - scoring pushes opponent blocks out!

## Scoring

| Action | Points |
|--------|--------|
| Block in goal | 3 |
| Goal control | 10 |
| 1 robot parked | 8 |
| 2 robots parked | 30 |

## Match Structure

- 15 sec autonomous
- 1:45 driver control

## Mechanisms

- **Intake:** Collect blocks
- **Conveyor:** Move blocks through robot
- **Releaser:** Score blocks into goals
- **Arm/Lever:** Pneumatic actuators
